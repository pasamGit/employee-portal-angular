# Employee Portal Angular (Demo)

Features included:
- Login & Register UI
- Employee list page
- Lazy loaded modules
- Environment configuration
- Auth service
- Employee service
- JWT interceptor
- Error interceptor
- Logging service
- Pagination example
- Simple caching with shareReplay

Run:

npm install
ng serve

API expected at: http://localhost:8080/api

Requirements:
- Node.js 18.13+ (Angular CLI requirement)
